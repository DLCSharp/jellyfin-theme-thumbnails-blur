# jellyfin-theme-thumbnails-blur

Jellyfin theme to blur episode thumbnails. Useful to prevent spoilers.

Tested on Jellyfin 10.11.7: web (Chromium & Firefox) and mobile.

## Supported pages

- Season episode list
- Show page "More from"
- Episode "More from"
- Library pages

## Usage

Add this to your Jellyfin custom CSS (Dashboard > Branding > Custom CSS):

```css
@import url('https://cdn.jsdelivr.net/gh/DLCSharp/jellyfin-theme-thumbnails-blur@main/theme.css');
```
